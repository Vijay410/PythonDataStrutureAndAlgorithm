## What is Array?
Array is a **linear data structure** that stores elements in **contiguous memory**.

**Types**:
1. Static Array (fixed size)
2. Dynamic Array (resizable – Python list)

> Note: Python uses **dynamic arrays**.

## Time Complexity
- Access → O(1)
- Search → O(n)
- Insert → O(n)
- Delete → O(n)
